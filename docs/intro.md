# About SodiumNodes
#### Welcome to the official SodiumNodes documentation!
![SodiumNodes banner](../static/img/general/SodiumNodes%20banner.png)
SodiumNodes, founded on the 8th of March, 2025, provides **fast, reliable, and affordable** game server hosting built for **performance and stability**, offering **modern hardware**, **optimized configurations**, and **clear resource limits** to ensure consistent uptime and smooth gameplay.
